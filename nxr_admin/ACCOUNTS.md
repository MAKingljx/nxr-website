## 角色权限说明

### Admin 角色
- ✅ 查看所有数据
- ✅ 创建新录入
- ✅ 编辑所有条目
- ✅ 审核条目（批准/拒绝）
- ✅ 导出到主数据库
- ✅ 导出Excel文件
- ✅ 管理上传文件

### Reviewer 角色
- ✅ 查看所有数据
- ✅ 创建新录入
- ✅ 编辑自己创建的条目
- ✅ 审核条目（批准/拒绝）
- ✅ 查看导出功能（部分限制）

## 使用说明

### 登录方式
1. 访问: http://localhost:8081/admin
2. 输入用户名和密码
3. 点击登录

### 用户界面
- 右上角显示当前用户名和角色徽章
- Admin角色显示绿色徽章
- Reviewer角色显示蓝色徽章

### 账号特点
1. **简单易记**：用户名和密码都很简单
2. **角色区分**：清晰的权限划分
3. **无需数据库**：硬编码在应用中，无需额外配置
4. **易于扩展**：可以随时添加更多账号

## 安全建议
1. 定期更改密码
2. 不要分享账号信息
3. 审核员账号只用于审核工作
4. 管理员账号妥善保管

## 扩展说明
如需添加更多账号，修改 `app_updated.py` 中的 `valid_accounts` 字典：
```python
valid_accounts = {
    'admin': {'password': 'nxr2026', 'role': 'admin'},
    'reviewer1': {'password': 'review123', 'role': 'reviewer'},
    'reviewer2': {'password': 'review456', 'role': 'reviewer'},
    # 添加更多账号...
}
```
